ARCHLINUXVN -- Vietnamese ArchLinux users

NAME

archlinuxvn - group of vietnamese archlinux users

DESCRIPTION

archlinuxvn is group of vietnamese archlinux users. We are living in Vietnam, using ArchLinux, and enjoying KISS.

OVERVIEW

We are running a mailing list at http://groups.google.com/group/archlinuxvn This list is serving generic discussion and helps. You are welcome to join the list as that helps to gather information about the community.

You can also meet us on the official IRC channel #archlinuxvn on irc.freenode.net. The list is quite active and has an archive at http://archlinuxvn.tuxfamily.org/irclogs/ (basic authentication is required.)

And this is our homepage, at http://archlinuxvn.tuxfamily.org/. If you like coding, you may be interested in our repositories at Github.

OPTIONS

To get helps, please join and send an message to the mailing list. You can also use the IRC channel. Or, just simply quit ArchLinux and use Ubuntu instead :) For exit status, see next section.

EXIT STATUS

  • 0 You are using ArchLinux
  • 1 You are a Gentoo user
  • 2 You are an Ubuntu guy
  • 255 Windows is the only thing in your world

ENVIRONMENT

  • LOCATION The only acceptable values are any places in Vietnam
  • SYSTEM The only acceptable value is ArchLinux
  • LANG Should be Vietnamese
